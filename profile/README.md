# Promethix Labs

Promethix Labs is an open-source engineering group focused on pragmatic platform tooling, .NET architecture, Kubernetes automation, and secure software delivery.

We build small, focused tools that solve real operational problems: database transaction boundaries, Kubernetes route automation, Helm packaging, release provenance, and infrastructure-adjacent developer workflows.

## Projects

### [Promethix.Framework.Ado](https://github.com/PromethixLabs/Promethix.Framework.Ado)

A lightweight ADO.NET and Dapper scope library for managing connection and transaction lifecycles without forcing repository code to pass database objects everywhere.

### [Promethix.Operator.CloudflareRouter](https://github.com/PromethixLabs/Promethix.Operator.CloudflareRouter)

A Kubernetes operator for publishing services through Cloudflare Tunnels and managing route lifecycle declaratively.

### [Promethix Helm Charts](https://github.com/PromethixLabs/charts)

Public Helm chart repository for Promethix Labs components.

## Engineering principles

- Explicit, reviewable architecture
- Production-grade defaults
- Security and operational safety first
- Supply-chain transparency
- Small, composable tools over large frameworks

## Supply-chain and release transparency

Where practical, Promethix projects publish checksums, SBOMs, and provenance material for release artefacts.
